Este código tiene problemas con los caracteres tildados, hay que buscar una solución para poder utilizarlo en algo más

Problema solucionado, la falla era causada porque se obtienen los datos del URL, y eso viene codificado. Al no haber nunca una decodificación javascript lo interpretaba mal. Al usar decodeURI en el valor que se va a imprimir se resuelve el problema.
